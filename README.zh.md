## 为什么必须重新学习"提问"

当代，大型语言模型已成为研究助手、写作工具、思维伙伴。人们用它来理解时事、分析趋势、探讨历史、整理思路、生成文本、获取建议。

由此产生一个看似简单的问题：**我们应该怎样向它提问？**

表面上看，这像是一个技术问题——prompt 写得更长、更详细、更"专业"，就能得到更好的答案？

但深入思考会发现，这不仅仅是技术问题。**它首先是语言问题，其次是知识论问题，最后也是伦理问题。**

---

## 大型语言模型究竟在做什么

语言模型看似在"回答问题"，但不是因为它直接感知现实，也不是因为它固有真理。

它的核心机制：**从文本中学习统计规律，然后预测最可能出现的下一步。**

它本质上是**概率性语言预测系统**——只是次要地表现出理解、解释、推理或创造的假象。

这意味着：
- 模型的答案通常是语言空间中的高概率产物
- 可能有用、流畅、接近现实
- 但并不等同于现实本身

---

## 提问不是中性的

不是所有问题都属于同一类型。同一话题可以在不同层次被提出：

**政治话题中：**
- "这项政策的主要内容是什么？" → 事实性
- "支持者和反对者如何解释它？" → 解释性
- "它可能产生什么影响？" → 预测性
- "应该支持它吗？" → 判断性

**经济话题中：**
- "当前通胀率是多少？" → 事实
- "为什么体感通胀与官方数据有差异？" → 分析
- "会发生衰退吗？" → 预测
- "我应该调整投资策略吗？" → 建议

**历史话题中：**
- "这个事件发生在何时？" → 事实
- "为什么会发生？" → 解释
- "如果历史走向不同会怎样？" → 反事实推测

**一个成熟的提问者首先要知道：自己要的是事实、解释、预测，还是建议。**

## 为什么精确提问很关键

许多人以为有效提问就是写得更长、更繁复、更"专业"。

错了。好的 prompt 不是更长，而是**更清楚；不是更繁复，而是更准确。**

精确提问包括：

1. **明确问题类型** – 事实、解释、预测还是建议？
2. **限定范围** – 哪个国家、时期、数据集、人物或时间框架？
3. **说明目标** – 是理解概念还是做决策？
4. **指定约束** – 基于哪些来源、标准或角度？
5. **区分层次** – 别把事实、观点、预测和价值判断混在一句话里

目的不是让模型更"听话"，而是**减少误解、减少猜测、减少语言上的漂移。**

## 语法和表达清晰度

语法不是装饰。它编码关系：

- 谁在做什么
- 哪个条件限制哪个结论
- 什么是因果、转折或假设关系
- 什么是核心，什么只是背景

**当语法不清楚时，模型必须自己补全缺失信息。** 它猜得越多，输出就越可能在语言上合理但与你的本意不符。

### 为什么训练过程中这很重要

说模型"只在语法正确的文本上训练"过于简化。更准确地说：

- 高质量预训练语料倾向包含更规范、易读的文本
- 但训练数据也包含口语、非标准表达、甚至错误文本
- 不过总体上，模型从相对规范的文本中学到了最稳定的句法和篇章模式

**语法之所以重要，不是因为模型"拒绝"错误语法，而是因为更清晰的表达更符合它学到的稳定模式。**

## 语义精确性

许多词看似人人都懂，但边界其实不清：

- 自由、公平、失败、危机
- 保守、激进、有效、靠谱

这些词在政治、经济、历史、日常讨论中都很常见，但含义**高度依赖语境。**

### 模糊词扩大答案空间

如果问"这个政策公平吗""这个措施有效吗""这个人物是好人吗"，模型无法确定你用的"公平""有效""好人"按什么标准定义。它只能在多个可能框架中做概率补完。

### 更好的做法：拆解抽象词

与其问：
> 这项政策公平吗？

不如问：
- 它对不同收入群体的影响是否相同？
- 谁承担成本，谁获得收益？
- 它符合法律意义上的平等，还是结果意义上的平等？

与其问：
> 有效吗？

不如问：
- 它是否达到政策声明中的目标？
- 短期有效还是长期有效？
- 是否伴随新的副作用？

**定义越清楚，答案就越容易被检验、比较和讨论。**

## Embedding 和表示学习

现代语言模型将词、短语、上下文映射到高维向量空间。这不是字典式定义，而是**从表达在各种语境中出现的方式学到的分布式表示。**

### 与清晰表达的关系

语义相关的表达在表示空间中聚集得更近——"通胀""物价上升""生活成本压力"可能靠在一起。但靠近不等于相同。同一个词在不同语境中含义会变化。

**模糊词激活的语义邻域太广。** 当你用"公平""失败""危险"这类词时，模型可能联想到很多相关但不完全一致的语义区域，导致它沿着常见用法、常见立场或常见叙述模板展开，而非精准对准你的本意。

**Embedding 不会"导致"模糊，但它帮助解释为什么模糊提问更容易跑偏。**

## 幻觉问题：双向的困境

"幻觉"（hallucination）常被理解为模型"瞎编"。

更深层的理解：模型生成的是**在语言上最像答案的内容，而不是必然与现实相符的内容。**

如果你的问题定义不清、证据不足、语义边界模糊，模型就会产生听起来合理但实际不可靠的回答。

**幻觉不只是模型的问题——往往也是提问方式失衡的结果。**

## 负责任地使用

善用语言模型，意味着拒绝滥用。

常见的滥用方式：

1. **预设结论，让模型包装** – "请证明这项政策肯定失败"
2. **把宣传伪装成分析** – "经济显然完了，帮我写得更有说服力"
3. **外包判断责任** – "别分析，直接告诉我谁对谁错"
4. **把模型当成最终权威** – 不再查证、比较、区分事实与推测

## 分层提问的方法

把复杂问题拆成有纪律的序列：

### 第一层：事实
哪些信息相对稳定、可以核实？

### 第二层：解释
不同视角如何解释这些事实？基于什么？

### 第三层：预测
如果条件继续演进，可能出现哪些不同结果？

### 第四层：建议
在不同目标和约束下，应该做什么？

这种分层**减少混淆和歧义。**

## 实际例子

### 政治
**较弱的提问：**
> 这个政策彻底失败了吗？媒体是不是都在带风向？

**更好的提问：**
1. 这项政策的主要内容是什么？
2. 支持者和反对者的核心理由是什么？
3. 有哪些公开数据或报道支持不同说法？
4. 按短期民意、长期财政影响、法律可行性等不同标准，它会如何被评价？

### 经济
**较弱的提问：**
> 经济这么差，我是不是应该马上卖掉所有股票？

**更好的提问：**
1. 哪些指标显示经济放缓？
2. 这些指标是一致还是矛盾？
3. 历史上类似条件下市场出现过哪些不同结果？
4. 不同风险承受能力对这种情况意味着什么？

### 历史
**较弱的提问：**
> 这个历史人物到底是好人还是坏人？

**更好的提问：**
1. 关于这个人物，哪些事实相对无争议？
2. 不同历史解释之间的分歧在哪里？
3. 各自依据什么史料和价值前提？
4. 为什么现代人喜欢用今天的标准评判过去的人物？

## 结语：提问工程就是自我训练

学会使用语言模型，不只是学一个新工具。

它是在重新学习：
- 如何把问题问清楚
- 如何区分事实与判断
- 如何减少语言中的歧义
- 如何避免把偏见伪装成"智能结果"

**语言模型不是神谕、不是裁判、也不是替代思考的捷径。**

把它想象成一面由语言和概率构成的镜子。你怎样提问，它就怎样反射。你怎样组织语言，它就怎样组织回答。你怎样处理事实、偏见和理性的关系，它就怎样把这些放大。

**真正高水平的提问工程，不是让机器更像人，而是让提问者自己变得更清楚、更诚实、更有条理。**
